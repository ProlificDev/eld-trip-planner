───────────────────────────────────────────────────────────────

KEY: OPENROUTE_API_KEY

   INSTRUCTIONS

1. Click "Add Environment Variable" button on Render
2. Copy KEY from above
3. Paste into KEY field
4. Copy VALUE from above
5. Paste into VALUE field
6. Repeat for each variable

For SECRET_KEY:
- Go to: https://djecrety.ir/
- Click "Generate"
- Copy the generated key
- Paste as VALUE
